1. Install dependencies from the studio package directory:
   ```bash
   cd apps/studio && pnpm add svelte-codemirror-editor @codemirror/lang-json @codemirror/lint paneforge jsonpos && pnpm add -D @codemirror/theme-one-dark
   ```

2. Create `apps/studio/src/lib/editor/useJsonSync.ts`:
   - `findNodeOffset(json: string, nodeUniqueId: string): { start: number; end: number } | null`
     - Parse json, find node index by unique-id, use jsonpos to get character offsets
   - `findRelationshipOffset(json: string, relUniqueId: string): { start: number; end: number } | null`
     - Same pattern for relationships array
   - Both functions return null on parse error or if ID not found

3. Create `apps/studio/src/lib/editor/CodePanel.svelte`:
   Props (Svelte 5 $props):
   - `value: string` — the CALM JSON string (bindable)
   - `onchange?: (value: string) => void` — called on every edit
   - `parseError?: string | null` — error message to show in status bar
   - `selectedNodeId?: string | null` — triggers scroll-to-node
   - `selectedEdgeId?: string | null` — triggers scroll-to-edge

   Structure:
   ```
   <div class="code-panel">
     <!-- Tab bar -->
     <div class="tab-bar">
       <button class="tab active">CALM JSON</button>
       <button class="tab disabled" title="Coming in Phase 5" disabled>calmscript</button>
       <!-- Status indicator: green dot when valid, red dot + message when parseError -->
       <span class="status">{parseError ? 'Invalid JSON' : 'Valid'}</span>
     </div>
     <!-- Editor -->
     <CodeMirror {value} lang={json()} {extensions} lineNumbers onchange={...} onready={...} />
   </div>
   ```

   Extensions (via $derived):
   - `linter(jsonParseLinter())` and `lintGutter()` for error squiggles
   - `oneDark` conditionally when `isDark()` is true (import from theme store)
   - `EditorView.lineWrapping` for long lines

   Selection scroll: Use `$effect` watching `selectedNodeId`/`selectedEdgeId`. When changed, call `findNodeOffset`/`findRelationshipOffset` to get offsets, then dispatch `view.dispatch({ selection: { anchor, head }, scrollIntoView: true })` on the stored EditorView ref (from `onready` callback).

   When CodeMirror `onchange` fires, call the `onchange` prop. Do NOT attempt to parse JSON here — the parent handles debouncing.

   Style: match existing dark/light theme patterns from the codebase. Tab bar is compact (28-32px height). Status indicator is small text right-aligned in tab bar.

   Add SPDX Apache 2.0 header comment at top of both files.
