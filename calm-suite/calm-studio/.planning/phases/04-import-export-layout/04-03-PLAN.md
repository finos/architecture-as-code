Modify `apps/studio/src/routes/+page.svelte` to integrate all Phase 4 features:

**1. Import new modules:**
- `import Toolbar from '$lib/toolbar/Toolbar.svelte'`
- `import { openFile, saveFile, saveFileAs } from '$lib/io/fileSystem'`
- `import { getFileName, getFileHandle, getIsDirty, markDirty, markClean, resetFileState } from '$lib/io/fileState.svelte'`
- `import { exportAsCalm, exportAsSvg, exportAsPng, exportAsCalmscript } from '$lib/io/export'`
- `import { resetModel } from '$lib/stores/calmModel.svelte'`
- `import { resetHistory } from '$lib/stores/history.svelte'`
- `import { onMount } from 'svelte'`

**2. Wire file operations:**
- `handleOpen()`: call `openFile()`, on success call `importCalmFile(result.content, result.name)` (already exists from Plan 01), then `markClean(result.name, result.handle)`
- `handleSave()`: get json via `getModelJson()`, call `saveFile(json, getFileHandle(), getFileName() ?? 'architecture.calm.json')`, on success `markClean()`
- `handleSaveAs()`: get json, call `saveFileAs(json, getFileName() ?? 'architecture.calm.json')`, on success `markClean(name, handle)`
- `handleNew()`: if `getIsDirty()`, use `window.confirm('You have unsaved changes. Continue?')` — if cancelled, return. Then `resetModel()`, `resetHistory()`, `resetFileState()`, set `nodes = []; edges = []`

**3. Wire export operations:**
- `handleExportCalm()`: call `exportAsCalm(getModelJson())`
- `handleExportSvg()`: call `exportAsSvg(nodes)`
- `handleExportPng()`: call `exportAsPng(nodes)`
- `handleExportCalmscript()`: call `exportAsCalmscript('')` (stub — empty string for now, Phase 5 will provide real content)

**4. Mark dirty on canvas mutations:**
- In existing `handleCodeChange` function (after successful parse/apply): add `markDirty()`
- In existing `handlePropertyMutation` function: add `markDirty()`
- After palette place, node drag, edge connect, undo/redo, paste — wherever nodes/edges are mutated, add `markDirty()`. The simplest approach: add an `$effect` that watches `nodes` and `edges` references and calls `markDirty()` (with a guard to skip the initial render).

**5. Add Toolbar to layout:**
- Place `<Toolbar>` above the existing `<PaneGroup>` in the template
- Pass all handler props and state: `onopen={handleOpen}`, `onsave={handleSave}`, etc.
- Pass `filename={getFileName()}` and `isDirty={getIsDirty()}`

**6. Add dismissible error banner for import errors (per user decision: "Invalid/malformed CALM JSON shows error toast, canvas unchanged"):**
- The `importError` $state variable already exists from Plan 01 (set when importCalmFile encounters invalid JSON)
- Add a conditional error banner in the template, positioned above the canvas area (below Toolbar, above PaneGroup):
  ```svelte
  {#if importError}
    <div class="error-banner">
      <span>{importError}</span>
      <button onclick={() => importError = null} aria-label="Dismiss error">x</button>
    </div>
  {/if}
  ```
- Style the error banner: full width, red/orange background (using CSS var or `#fef2f2` light / `#451a1a` dark), padding 8px 16px, flex row with space-between, dismiss X button on the right
- Banner auto-dismisses: NOT needed — user clicks X or it clears on next successful import (importError = null in importCalmFile success path)

**7. Add keyboard shortcuts:**
- Use `onMount` to add a `keydown` listener on `window`:
  - `Cmd+O` (metaKey + key === 'o'): `e.preventDefault(); handleOpen()`
  - `Cmd+S` (metaKey + key === 's' + !shiftKey): `e.preventDefault(); handleSave()`
  - `Cmd+Shift+S` (metaKey + shiftKey + key === 's'): `e.preventDefault(); handleSaveAs()`
  - `Cmd+N` (metaKey + key === 'n'): `e.preventDefault(); handleNew()`
- Clean up listener in `onMount` return function
- Remove the temporary Cmd+O handler from Plan 01 if it was added inline

**8. Add beforeunload guard:**
- In `onMount`: add `window.addEventListener('beforeunload', handler)` where handler calls `e.preventDefault(); e.returnValue = ''` if `getIsDirty()` is true
- Clean up in return function

**9. Update document title reactively:**
- Add `$effect` that sets `document.title` based on `getFileName()` and `getIsDirty()`:
  - Dirty: `"${filename} \u2022 CalmStudio"` (bullet dot)
  - Clean: `"${filename} - CalmStudio"`
  - No file: `"CalmStudio"`
