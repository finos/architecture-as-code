## Decisions Made

- `aigf-rules.ts` imports only from `@calmstudio/calm-core` — pure TypeScript ensures vitest testability without Svelte transform setup
- Data-classification badge uses inline style string (not Svelte class binding) — avoids scoped CSS limitations when color values are dynamic
- `ControlsList` collapsed by default — most nodes don't have controls, avoids visual noise
- `aigf-010` always fires as `info` severity — governance score report is always useful regardless of AI node count
- `aigf-006` reports per-agent (one issue per agent without isolation) — enables precise node navigation from ValidationPanel badge clicks
