## Decisions Made

- Full-screen modal for template picker: maximizes visual impact for OSFF Toronto demo (April 13-14, 2026)
- `initAllTemplates()` mirrors `initAllPacks()` pattern with synchronous static imports — no dynamic import complexity
- `loadTemplate()` uses `const { _template, ...arch } = t; return arch` — clean destructuring, no mutation of registry data
- Export stripping added to `exportAsCalm()` directly — ensures clean output regardless of how template was loaded
- Category tab labels: `fluxnova` → `FluxNova`, `ai-governance` → `AI Governance`, `general` → `General`
