## Decisions Made

- `{@const}` inside a `<div>` within `{#each}` is invalid in Svelte 5 — replaced with direct function calls in `{#if}` and `{#each}` expressions (discovered during build)
- `generateAIGFDecorator()` in `export.ts` has no store dependency — same logic duplicated standalone to keep export.ts pure and testable (consistent with aigf-rules.ts pattern)
- Tab reset implemented via `$effect` tracking `prevSelectionKey` — Svelte 5 idiomatic way to trigger reset when a derived value changes
