       - `pushSnapshot(nodes, edges)`: deep-clone nodes/edges arrays, drop future history (slice to pointer+1), push, increment pointer. CRITICAL: snapshot BEFORE mutation, not after (RESEARCH Pitfall 6).

       - `redo()`: returns next snapshot (pointer+1) or null if at end
       - `canUndo` and `canRedo` derived getters

       - Unlimited history (no cap — per CANV-05 requirement)

    4. Create `clipboard.svelte.ts` — copy/paste with new unique-ids:
       - `let clipboardNodes = $state<Node[]>([]);`
       - `copy(nodes: Node[])`: filter to `selected` nodes, store deep clones
       - `paste(existingNodes: Node[])`: returns new Node[] array — cloned clipboard nodes with `nanoid()` ids, `nanoid()` for `data.calmId`, position offset (+20, +20), `selected: false`. Returns the new nodes to be appended.
       - `hasClipboard` derived getter
       - Import `nanoid` for ID generation
       - Run tests — they should PASS (GREEN phase)

    5. Create `theme.svelte.ts` — dark mode with system preference detection:
       - `let isDark = $state(false);`
       - `initTheme()`: check localStorage('calmstudio-theme'), fall back to `matchMedia('(prefers-color-scheme: dark)')`, set `isDark`, toggle `dark` class on `document.documentElement`, set up `matchMedia` change listener
       - `toggleTheme()`: flip `isDark`, update localStorage, toggle `dark` class on `<html>`
       - Listen to system preference changes via `matchMedia.addEventListener('change', ...)`
       - Export `isDark`, `initTheme`, `toggleTheme`
