  <action>
    Create one Svelte component per node type. Each component:
    - Receives `NodeProps` via `let { id, data, selected }: NodeProps = $props();`
    - Includes `<NodeResizer>` (visible when selected)
    - Has 4 `<Handle>` components (top=target, bottom=source, left=target, right=source) as default handles
    - If `data.interfaces` array exists, render additional named handles with `id={iface['unique-id']}` positioned along node edges
    - Renders a distinct SVG shape + name label + small type icon
    - Uses monochrome styling — shape alone differentiates (per user decision: NO per-type coloring)
    - All styling via Tailwind utility classes + inline SVG

    **Shape assignments (per user decision — distinct shapes like AWS icons):**
    - `actor`: Person/stick figure silhouette (head circle + body trapezoid)
    - `system`: Large rectangle with double border (system boundary box)
    - `service`: Rounded rectangle with gear/cog icon
    - `database`: Cylinder (ellipse top + rect body + ellipse bottom)
    - `network`: Cloud shape (overlapping circles)
    - `webclient`: Browser window shape (rectangle with top toolbar dots)
    - `ecosystem`: Hexagon shape (external system boundary)
    - `ldap`: Shield/key shape (directory service)
    - `data-asset`: Document shape (rectangle with folded corner)
    - `generic`: Plain rectangle with dashed border + custom type label (fallback for unknown types)
    - `container`: Large labeled boundary box with header bar showing name, collapse/expand toggle button, min-width/min-height for children. Use `style="width: {width}px; height: {height}px;"` for resizable parent bounds.

    **ContainerNode specifics:**
    - Has a collapse toggle (chevron icon in header)
    - When collapsed: renders as compact node (small rect with name only)
    - When expanded: renders as boundary box — children positioned inside (Svelte Flow handles this via `parentId` + `extent: 'parent'`)
    - `data.collapsed` boolean controls state
    - On collapse toggle, dispatch a custom event or update node data so the canvas can hide/show children

    **Node data shape convention:**
    ```typescript
    data: {
      label: string;            // Display name
      calmId: string;           // CALM unique-id
      calmType: string;         // CALM node-type
      interfaces?: CalmInterface[];
      collapsed?: boolean;      // ContainerNode only
    }
    ```

    All files MUST include Apache 2.0 SPDX headers.
