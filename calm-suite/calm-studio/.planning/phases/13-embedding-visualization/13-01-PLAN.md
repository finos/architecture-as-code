<objective>
Create the `<calm-diagram>` web component package that renders any CALM JSON architecture as an interactive SVG diagram using ELK.js layout with full extension pack icon/color support.

Purpose: Enables any developer to embed a CALM architecture diagram in any webpage or framework with a single HTML tag and zero configuration. This is the primary distribution mechanism for CALM visualization beyond the studio app.

Output: `packages/web-component/` package producing `@calmstudio/diagram` with IIFE (CDN) and ESM (npm) bundles, plus `CalmFlow`/`CalmTransition` types added to calm-core.
</objective>
