    - create_architecture: overwrites existing file content
    - describe_architecture: reads .calm file and returns structured summary (node count, relationship count, node names/types)
    - add_node: appends node to .calm file, returns confirmation with node ID
    - get_node: returns node details by unique-id, or error if not found
    - update_node: modifies node fields by unique-id (merge, not replace), writes back
    - delete_node: removes node AND all relationships referencing it, returns confirmation
    - query_nodes: filters by type and/or name substring, returns matching node summaries
    - batch_create_nodes: adds multiple nodes in one call, returns count
  </behavior>

   - Export `registerArchitectureTools(server: McpServer)` function
   - Register `create_architecture` tool: accepts CreateArchitectureSchema, builds CalmArchitecture from input, calls writeCalmFile, returns toolSuccess with summary (node count, rel count, file path, `calmstudio://open?file=...` deep link)
   - Register `describe_architecture` tool: accepts DescribeArchitectureSchema, calls readCalmFile, returns structured text summary: node count, relationship count, list of node names with types, list of relationships (source->destination, type)
   - All responses use toolSuccess/toolError helpers. Keep responses concise (summary + IDs, not full JSON per CONTEXT.md decision).
   - SPDX header.

2. Create `packages/mcp-server/src/tools/nodes.ts`:

   - Register tools: `add_node`, `get_node`, `update_node`, `delete_node`, `query_nodes`, `batch_create_nodes`
   - All follow read-mutate-write pattern: `const filePath = resolveFile(args.file); const arch = readCalmFile(filePath); /* mutate */ writeCalmFile(filePath, arch);`
   - `delete_node` also removes relationships where `source === id || destination === id`
   - `query_nodes` filters by `node-type` and/or name (case-insensitive substring match)
   - `batch_create_nodes` pushes all nodes in single read-write cycle
   - Errors: node not found returns toolError with available IDs
   - SPDX header.
