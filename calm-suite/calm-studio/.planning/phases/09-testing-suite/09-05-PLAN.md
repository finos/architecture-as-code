  <files>.github/workflows/ci.yml</files>
  <action>
    Read the current .github/workflows/ci.yml.

    1. In the existing `build-lint-test` job, replace the Test step (`pnpm -r run test`) with a coverage-enabled test step:
       ```yaml
       - name: Test with coverage
         run: |
           pnpm --filter @calmstudio/calm-core run test:coverage
           pnpm --filter @calmstudio/extensions run test:coverage
           pnpm --filter @calmstudio/mcp-server run test:coverage
           pnpm --filter @calmstudio/studio run test:coverage
       ```
       This runs vitest with --coverage flag, which enforces the thresholds configured in each vitest config. If coverage falls below thresholds, the step fails.

    2. Add coverage artifact upload step after the test step:
       ```yaml
       - name: Upload coverage reports
         uses: actions/upload-artifact@v4
         if: always()
         with:
           name: coverage-reports
           path: |
             packages/calm-core/coverage/
             packages/extensions/coverage/
             packages/mcp-server/coverage/
             apps/studio/coverage/
           retention-days: 14
       ```

    3. Add a new job `e2e-tests` that runs on merge to main only (not on PRs — per CONTEXT.md discretion, E2E on merge-to-main initially):
       ```yaml
       e2e-tests:
         name: E2E Tests
         runs-on: ubuntu-latest
         if: github.event_name == 'push' && github.ref == 'refs/heads/main'
         needs: build-lint-test
         steps:
           - name: Checkout
             uses: actions/checkout@v4

           - name: Setup pnpm
             uses: pnpm/action-setup@v4
             with:
               version: 9

           - name: Setup Node.js
             uses: actions/setup-node@v4
             with:
               node-version: '20'
               cache: 'pnpm'

           - name: Install dependencies
             run: pnpm install --frozen-lockfile

           - name: Install Playwright browsers
             run: npx playwright install --with-deps chromium

           - name: Build
             run: pnpm -r run build

           - name: Run E2E tests
             run: pnpm --filter @calmstudio/studio run test:e2e
       ```

    4. Preserve all existing jobs (reuse-compliance, cve-scan, commitlint) unchanged.
    5. Preserve the SPDX header at the top of ci.yml.
    6. Keep permissions: contents: read.
