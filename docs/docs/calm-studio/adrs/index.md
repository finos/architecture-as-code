# Architecture Decision Records (ADRs)

Architecture Decision Records document significant design choices made during CalmStudio development.
Each ADR follows the [MADR 4.0](https://adr.github.io/madr/) format: Title, Status, Context, Decision, Consequences.

ADRs are being added in Phase 10 Plan 03.
