# Operations Guides

These guides help you create operations documentation from your CALM architectures.

## Available Guides

- [**Operations Advisor**](ops-advisor) - Use AI to generate operations documentation
- [**Create Operations Docs**](ops-docs) - Build runbooks, playbooks, and deployment guides

## Prerequisites

Before using these guides, you should be comfortable with:
- Creating CALM architectures
- Using the CALM CLI and docify
- Basic operations/SRE concepts

If you need to learn CALM basics, start with the [Beginner Tutorials](../../tutorials/beginner/).
