# Documentation Guides

These guides help you generate and customize documentation from your CALM architectures.

## Available Guides

- [**Generate Documentation**](docify) - Use docify to create Markdown docs from architecture data
- [**Create Custom Templates**](custom-widgets) - Build templates using CALM widgets
- [**Advanced Handlebars Templates**](handlebars) - Full control with raw Handlebars syntax
- [**AI Architecture Advisor**](ai-advisor) - Use AI to analyze and improve architectures

## Prerequisites

Before using these guides, you should be comfortable with:
- Creating CALM architectures with nodes and relationships
- Basic Markdown syntax
- Using the CALM CLI or VSCode extension

If you need to learn these basics, start with the [Beginner Tutorials](../../tutorials/beginner/).
