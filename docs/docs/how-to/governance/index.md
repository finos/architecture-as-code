# Governance Guides

These guides help you implement architecture governance with patterns, standards, and validation.

## Available Guides

- [**Create Patterns**](patterns) - Define reusable architecture templates
- [**Define Standards**](standards) - Create organizational validation rules
- [**Multi-Pattern Validation**](multi-pattern-validation) - Validate against multiple patterns

## Prerequisites

Before using these guides, you should be comfortable with:
- Creating complete CALM architectures
- Understanding JSON Schema basics
- Using the CALM CLI for validation

If you need to learn CALM basics, start with the [Beginner Tutorials](../../tutorials/beginner/).
