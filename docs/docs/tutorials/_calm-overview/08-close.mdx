  <aside className="notes">
    Four tiers, pick your entry point.
    Beginner: install CLI, write your first CALM file, validate it. Should take 20–30 minutes.
    Intermediate: assumes tools are installed. Focus is on patterns, customizations/standards,
    and running docify to generate a documentation site.
    Practitioner: AI-assisted architectural modelling — participants use the CALM AI tools to model
    a real system from the architect's perspective. Great for people who already know the basics.
    Advanced (QCon demo): the full five-scenario end-to-end walkthrough. Requires a Kubernetes
    cluster (see conferences/qcon). Plan 30–60 minutes for the full flow.
    Encourage people to pair up — the Slack #calm channel is there if they get stuck.
  </aside>
