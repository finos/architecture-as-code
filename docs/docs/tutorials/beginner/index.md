## Prerequisites

Before starting, ensure you have:

- **Git** installed
- **Node.js** (20+) and npm installed
- **VSCode** editor (version 1.96+)
- **GitHub account** (recommended for portfolio sharing)
- **GitHub Copilot** access (optional but recommended)

## Time Commitment

Each tutorial takes approximately **1-2 hours** to complete. You can work through them at your own pace.
