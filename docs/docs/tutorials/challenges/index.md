## How Challenges Work

Each challenge places you in a realistic scenario:

1. **Context** - You're given a role and situation
2. **Requirements** - Specific deliverables to produce
3. **Validation** - Checklist to verify completion
4. **Reflection** - Questions to consolidate learning
